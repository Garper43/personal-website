Gradient is a minecraft plugin that helps you make gradients Usage You start by building the shape that you want to paint into a gradient. You then use the following command to create a gradient: /grad <pattern> <blur> [drippage] [-i] <pattern> sets the blocks/colors that will be used in the gradient <blur> decides how much the pattern will be blurred [drippage] decides how much each vertical splice can be randomly shifted vertically Here you can see what it looks like if you combine <blur> and [drippage]: Technicalities Here is the formula that is being used to determine what block to place in a certain place: Where i is the index of the block in the <pattern> set M is the <pattern> set y is the relative y coordinate of the block relative to the selected area h is the height of the selected area s is the randomly chosen vertical offset based on [drippage] value b is the <blur> value r is a random number in the interval of [-0.5, 0.5)
